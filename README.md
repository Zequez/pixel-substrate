# Pixel Substrate

![Screenshot](/screenshot.png?raw=true "Screenshot")

An infinite-band pixel world engine.

In principle a calm computing experience designed in a way that a 4 year old toddler can
enjoy a safe creative digital space.

You may create infinite canvas-based pixel drawings that loop in the end. And then you may
also activate minigames that take that image created and turn it into the scenario for games.

If you wish to create a mini-game, clone this repo and make a PR.

## Running From the Web

Find built version at:

https://pixel-substrate.netlify.app

## Running Locally

How to run locally using [Bun](https://bun.com/) and [sitio](https://github.com/zequez/sitio):

```
git clone https://github.com/zequez/pixel-substrate
cd pixel-substrate
bun i -g @ezequielschw/sitio
sitio dev
```
